Project description

This game is a personalized twist on classic Tic-Tac-Toe, played on a 5×5 grid where players use first characters from their first names instead of X’s and O’s, transforming each match into a unique signature duel. The objective is to align at least four letters in a row—vertically, horizontally, or diagonally—to win. By replacing generic symbols with name-based tokens, the game adds identity and flair to each move, blending the simplicity of Tic-Tac-Toe with the strategic depth of Gomoku. It’s a playful yet thoughtful reimagining that turns every match into a reflection of the players themselves.
